# Ejecutar en Desarrollo

1. Instalar NestJs CLI
```
npm i -g @nestjs/cli
```
2. Clonar el repositorio
3. Ejecutar 
```
npm i / yarn install
```
4. Iniciar Base de Datos
```
docker compose up -d
```

5. Insertar Semillas en la Base de Datos
```
http://localhost:3000/api/seed
```

## Stack Utilizado
* Docker
* MongoDB
* NestJs
